# CAFE_FAH

Cafe chatbot project with:

- Supabase Postgres + pgvector
- FastAPI backend for RAG and chat APIs
- Vite + React frontend
- Gemini 2.5 Flash on the backend

## Deploy

- Backend: Render
- Frontend: Vercel
- Database: Supabase

See `DEPLOY.md` for the deployment steps.

## Supabase Quick Start

Run `infra/supabase_setup.sql` in the Supabase SQL Editor to create tables, enable pgvector, and seed starter cafe data.
